# Statful monetization ideas
## Long-term monetization
### One time fee for an agent purchase
The customer pays a one time fee when buying a new agent.
### Running costs
The customer pays for the running of the agents in terms of real time or in agent token use.
### Agent description limit extension fee
The customer pays a fee for writing longer agent description in the agent's job description.
## Initial custimer fees
### None
The customer does not pay anything to start their company, and they get a basic starting agent for free, though the agent is limited in what they can do.
### No free agents
The customer has to pay to get started.
