# 5-minute iteration heartbeat

On each heartbeat run:
1) Evaluate current project state (progress/blockers/gaps).
2) Perform one concrete work step.
3) Summarize updates in relevant files:
   - agent_instructions/everyone/ideers.md
   - agent_instructions/everyone/todo.md
   - agent_instructions/everyone/completed.md
   - memory/YYYY-MM-DD.md

Keep entries concise and precise.
